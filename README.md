# oow-bot
Telegram bot for best team ever - OOW

Prerequisites
-------------
Required software:
- Ruby `>= 2.6.0`

Install
-----------------
```
git clone https://github.com/olehvavryniv/oow-bot.git && cd oow-bot
bundle install
```
Create `.ENV` file from `.env.example` or ask some developers for example

`cp .env.example .env`

Launch
------------
```
ruby oow-bot.rb
```
